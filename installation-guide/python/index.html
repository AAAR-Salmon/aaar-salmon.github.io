Python開発環境クイックインストールガイド

挨拶

新入生の皆さん、及び新入生ではない皆さん、ご入学おめでとうございます。 Pythonを始めようという心意気にまず賛辞を贈ります。Pythonはいいぞ。

Pythonがどんな言語かと言えば、ライブラリが豊富で機械学習に強い言語です。 まあ私はAI開発にはまだ手を出してないんですけど。 機械学習以外のライブラリもたくさんあります。

とにかくなんでもサクッとお手軽に書ける言語です。インストールもサクッとやっちゃいましょう。

Windows

Python公式ページのDownloadsからインストーラをダウンロードして、成り行きに任せてインストールします。

Mac

例によってPython公式ページのDownloadsからpkgファイルを落としてきて、成り行きに任せてインストールします。

Linux

例えばUbuntuやDebianなら、 

sudo apt install python3
でインストール出来ます。あれ、Windowsより簡単じゃね？

テスト

終わったら、コマンドプロンプトを立ち上げて、pythonと入力し(Ubuntuの場合python3)、Enterを押します。 Pythonが起動したら適当に3*4とか入力してください。計算結果が帰ってきたら成功です。

ソースファイルでの実行もテストしてみましょう。 とりあえず湯婆婆と契約でも結んでください。 ダウンロードして、python yubaba.py
